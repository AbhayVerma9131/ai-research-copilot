# AI Research Copilot

## 📌 Problem
Understanding research papers is time-consuming and complex.

## 🧠 Solution
Built an AI system that extracts insights from PDFs using NLP and transformer models.

## 🚀 Features
- PDF text extraction
- Automatic summarization
- Question answering system

## ⚙️ Tech Stack
Python, Transformers, Streamlit, PyMuPDF

## 🔬 Approach
- Extract text from PDF
- Use transformer models for summarization
- Implement QA system using NLP

## 🚀 Future Work
- Add semantic search (FAISS)
- Multi-document support
- Improve model accuracy
